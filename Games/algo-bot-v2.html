📐 Règles plateau

  1. Chaque joueur reçoit ses tuiles selon le niveau.
  2. Place les tuiles dans ta ligne de commande (une rangée de cases).
  3. Exécute les tuiles de gauche à droite en déplaçant ton pion.
  4. Atteins la case verte (sortie) 🟢 pour gagner !
  5. Mur ou hors-plateau → crash, on recommence.
  6. Les tuiles jouées restent en place entre les tours.

🤖 But du jeu

Programme ALGO-BOT pour atteindre la sortie 🟢 en plaçant des tuiles-instructions dans la ligne de commande.

🔁 Boucles

LOOP×N répète les N tuiles qui précèdent dans la même ligne. Exemple : ⬆️ ⬆️ LOOP×2 → avance 4 fois au total. Pour les boucles imbriquées, enchaîne plusieurs LOOP.

📦 Fonctions (niveaux avancés)

DEF F1 : place des tuiles dans la zone F1 pour définir la fonction.
CALL F1 : appelle la fonction depuis le programme principal. Le robot exécute toutes les tuiles de F1, puis reprend là où il était.

🔀 Conditions

SI 💎 : exécute la tuile suivante seulement si le robot est sur un cristal.
SI 🪨 : exécute la tuile suivante seulement si un mur est devant le robot.

⬆ Flèches (cases spéciales)

Certains niveaux ont des cases-flèche qui modifient la direction du robot quand il passe dessus.

⭐ Score

3 étoiles = programme minimal. 2 étoiles = acceptable. 1 étoile = tu as réussi mais il y a mieux !
