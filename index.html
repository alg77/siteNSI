À Propos

Nous présentons ici les projets et les élèves de la Spécialité NSI (Numérique et Sciences de l'Informatique) au Lycée St Laurent en classe de Première.

Langages utilisés

Nous travaillons sur le langage Python principalement, nous avons également appris le HTML, le CSS et un peu de JavaScript, notamment pour ce magnifique site !

Responsive

Ce site est responsive, vous pouvez surfer sur ordinateur, tablette ou smartphone librement.

Environnements de programmation

En NSI, nous utilisons des environnements de programmation variés tels que EduPython et Thonny pour Python, Visual Studio Code et SublimeText pour le développement web, et des plateformes en ligne comme Replit pour travailler sur des projets collaboratifs.
