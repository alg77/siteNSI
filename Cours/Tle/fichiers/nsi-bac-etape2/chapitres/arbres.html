Arbres

Arbres binaires, ABR, tas, parcours

Récursion Parcours O(log n)

🎯 Introduction

Un arbre est une structure de données hiérarchique composée de nœuds reliés par des arêtes. Les arbres binaires (chaque nœud a au plus deux fils) sont fondamentaux en informatique.

✅ Ce qu'il faut savoir pour le bac

  • Vocabulaire : racine, feuille, nœud, hauteur, profondeur, taille
  • Les trois parcours en profondeur : préfixe, infixe, postfixe
  • Le parcours en largeur (BFS)
  • Propriétés d'un ABR (Arbre Binaire de Recherche)
  • Recherche, insertion dans un ABR : O(h) = O(log n) si équilibré
  • Implémenter les parcours récursivement en Python

📖 Vocabulaire essentiel

TermeDéfinition
RacineNœud sans parent (sommet de l'arbre)
FeuilleNœud sans fils (enfants)
Nœud interneNœud avec au moins un fils
HauteurLongueur du plus long chemin racine → feuille
ProfondeurDistance d'un nœud à la racine
TailleNombre total de nœuds
Sous-arbre gauche/droitArbre enraciné au fils gauche/droit
💡

Formules utiles

Un arbre binaire parfait de hauteur h a 2^(h+1) - 1 nœuds et 2^h feuilles.

La hauteur minimale d'un arbre binaire de n nœuds est ⌊log₂(n)⌋.

🔄 Parcours d'arbres

class Noeud:
    def __init__(self, val, g=None, d=None):
        self.val = val; self.gauche = g; self.droite = d

# Parcours PRÉFIXE (racine → gauche → droite)
def prefixe(n):
    if n is None: return
    print(n.val)          # racine d'abord
    prefixe(n.gauche)
    prefixe(n.droite)

# Parcours INFIXE (gauche → racine → droite)
def infixe(n):
    if n is None: return
    infixe(n.gauche)
    print(n.val)          # racine au milieu
    infixe(n.droite)

# Parcours POSTFIXE (gauche → droite → racine)
def postfixe(n):
    if n is None: return
    postfixe(n.gauche)
    postfixe(n.droite)
    print(n.val)          # racine en dernier

# Parcours en LARGEUR (BFS)
from collections import deque
def bfs(racine):
    if not racine: return
    file = deque([racine])
    while file:
        n = file.popleft()
        print(n.val)
        if n.gauche: file.append(n.gauche)
        if n.droite: file.append(n.droite)
💡

Moyen mnémotechnique

Préfixe → Précède (racine d'abord) | Infixe → entre les deux | Postfixe → Postérieur (racine après)

L'infixe d'un ABR donne les valeurs triées en ordre croissant !

🔍 Arbre Binaire de Recherche (ABR)

Un ABR vérifie la propriété : pour tout nœud, toutes les valeurs du sous-arbre gauche sont inférieures à la valeur du nœud, et toutes les valeurs du sous-arbre droit sont supérieures.

def recherche_abr(n, cible):
    if n is None: return False
    if n.val == cible: return True
    elif cible < n.val: return recherche_abr(n.gauche, cible)
    else: return recherche_abr(n.droite, cible)

def inserer_abr(n, val):
    if n is None: return Noeud(val)
    if val < n.val: n.gauche = inserer_abr(n.gauche, val)
    elif val > n.val: n.droite = inserer_abr(n.droite, val)
    return n

def hauteur(n):
    if n is None: return -1  # convention : hauteur arbre vide = -1
    return 1 + max(hauteur(n.gauche), hauteur(n.droite))
🚨

ABR dégénéré

Si on insère des valeurs déjà triées (1, 2, 3, 4...), l'ABR devient une liste chaînée de hauteur n → la recherche est O(n), pas O(log n) !

⛰️ Tas (Heap)

Un tas binaire est un arbre binaire complet qui respecte la propriété de tas : chaque nœud est plus grand (tas-max) ou plus petit (tas-min) que ses fils.

import heapq  # implémente un tas-min en Python

tas = []
heapq.heappush(tas, 5)
heapq.heappush(tas, 2)
heapq.heappush(tas, 8)

minimum = heapq.heappop(tas)   # retire le minimum : 2

# Créer un tas depuis une liste existante
lst = [3, 1, 4, 1, 5]
heapq.heapify(lst)              # O(n)
🚨

Pièges fréquents au bac

  • Confondre hauteur (chemin le plus long) et profondeur (distance à la racine)
  • Oublier le cas de base if n is None: return dans un parcours récursif
  • L'infixe d'un ABR donne une liste triée — utile pour vérifier qu'un arbre est bien un ABR
  • Un ABR avec doublons : convention à préciser (souvent val <= n.val va à gauche)

🌲 Arbres binaires — récursivité

⭐ Facile

Exo A1 — Taille d'un arbre

Écrire une fonction récursive taille(n) qui retourne le nombre de nœuds d'un arbre binaire.

def taille(n):
    if n is None: return 0
    return 1 + taille(n.gauche) + taille(n.droite)
# Complexité : O(n) — on visite chaque nœud une fois
⭐⭐ Intermédiaire

Exo A2 — Hauteur d'un arbre

Écrire hauteur(n) qui retourne la hauteur de l'arbre. Convention : arbre vide → -1, feuille → 0.

def hauteur(n):
    if n is None: return -1
    return 1 + max(hauteur(n.gauche), hauteur(n.droite))
# Feuille : max(-1,-1)+1 = 0 ✓
⭐⭐⭐ Avancé

Exo A3 — Comptage de feuilles

Écrire nb_feuilles(n) qui compte les feuilles (nœuds sans enfant).

def nb_feuilles(n):
    if n is None: return 0
    if n.gauche is None and n.droite is None:
        return 1  # c'est une feuille
    return nb_feuilles(n.gauche) + nb_feuilles(n.droite)
⭐⭐⭐⭐ Niveau bac

Exo A4 — Parcours en largeur (BFS)

Écrire bfs(racine) qui retourne la liste des valeurs de l'arbre en parcours en largeur (niveau par niveau).

from collections import deque

def bfs(racine):
    if not racine: return []
    file = deque([racine])
    resultat = []
    while file:
        n = file.popleft()
        resultat.append(n.val)
        if n.gauche: file.append(n.gauche)
        if n.droite: file.append(n.droite)
    return resultat
⭐⭐⭐⭐⭐ Expert

Exo A5 — Miroir d'un arbre

Écrire miroir(n) qui transforme l'arbre en son miroir (échange récursivement gauche et droite).

def miroir(n):
    if n is None: return None
    n.gauche, n.droite = miroir(n.droite), miroir(n.gauche)
    return n
# Remarque : l'infixe du miroir est l'infixe original inversé

🔍 ABR — Arbre Binaire de Recherche

⭐ Facile

Exo B1 — Recherche dans un ABR

Écrire recherche_abr(n, val) qui retourne True si val est dans l'ABR, False sinon. Complexité : O(h).

def recherche_abr(n, val):
    if n is None: return False
    if val == n.val: return True
    if val < n.val: return recherche_abr(n.gauche, val)
    return recherche_abr(n.droite, val)
# On élimine la moitié de l'arbre à chaque étape → O(log n) si équilibré
⭐⭐ Intermédiaire

Exo B2 — Insertion dans un ABR

Écrire inserer_abr(n, val) qui insère val dans l'ABR et retourne la racine.

class Noeud:
    def __init__(self, val):
        self.val = val; self.gauche = self.droite = None

def inserer_abr(n, val):
    if n is None: return Noeud(val)
    if val < n.val:
        n.gauche = inserer_abr(n.gauche, val)
    elif val > n.val:
        n.droite = inserer_abr(n.droite, val)
    return n  # val == n.val : pas de doublon
⭐⭐⭐ Avancé

Exo B3 — Liste triée depuis ABR

Écrire abr_vers_liste(n) qui retourne la liste triée de tous les éléments de l'ABR. Astuce : utiliser le parcours infixe.

def abr_vers_liste(n):
    if n is None: return []
    return (abr_vers_liste(n.gauche) +
            [n.val] +
            abr_vers_liste(n.droite))
# Parcours infixe (G→R→D) d'un ABR donne toujours les éléments triés !
⭐⭐⭐⭐ Niveau bac

Exo B4 — Vérifier qu'un arbre est un ABR

Écrire est_abr(n, mini=None, maxi=None) qui vérifie qu'un arbre binaire respecte la propriété ABR.

def est_abr(n, mini=None, maxi=None):
    if n is None: return True
    if mini is not None and n.val <= mini: return False
    if maxi is not None and n.val >= maxi: return False
    return (est_abr(n.gauche, mini, n.val) and
            est_abr(n.droite, n.val, maxi))
⭐⭐⭐⭐⭐ Expert

Exo B5 — Supprimer dans un ABR

Écrire supprimer_abr(n, val) qui supprime val de l'ABR. Cas à gérer : nœud feuille, 1 enfant, 2 enfants (remplacer par le successeur infixe).

def min_abr(n):
    while n.gauche: n = n.gauche
    return n.val

def supprimer_abr(n, val):
    if n is None: return None
    if val < n.val:
        n.gauche = supprimer_abr(n.gauche, val)
    elif val > n.val:
        n.droite = supprimer_abr(n.droite, val)
    else:  # trouvé
        if not n.gauche: return n.droite
        if not n.droite: return n.gauche
        # 2 enfants : remplacer par successeur infixe (min du sous-arbre droit)
        n.val = min_abr(n.droite)
        n.droite = supprimer_abr(n.droite, n.val)
    return n

📌 Fiche synthèse

PARCOURS

  • Préfixe : R → G → D
  • Infixe : G → R → D
  • Postfixe : G → D → R
  • Largeur : file

ABR

  • gauche < racine < droite
  • Recherche : O(h)
  • Équilibré : h = O(log n)
  • Infixe → trié

FORMULES

  • Arbre vide : hauteur = -1
  • Feuille : hauteur = 0
  • Nœuds parfait h : 2^(h+1)-1
  • heapq → tas-min Python
