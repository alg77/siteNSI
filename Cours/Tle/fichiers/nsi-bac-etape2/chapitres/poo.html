Classes Héritage Encapsulation

🎯 Introduction

La POO est un paradigme qui organise le code autour d'objets combinant des données (attributs) et des comportements (méthodes). Python est un langage multi-paradigme orienté objet.

✅ Ce qu'il faut savoir

  • Créer une classe avec __init__, self, attributs et méthodes
  • Héritage : class Enfant(Parent), super()
  • Polymorphisme : même méthode, comportement différent selon le type
  • Encapsulation : convention _attribut (protégé), __attribut (privé)
  • Méthodes spéciales : __str__, __repr__, __len__, __eq__

🏗️ Classes et objets

class Voiture:
    # Attribut de classe (partagé par toutes les instances)
    nb_roues = 4

    def __init__(self, marque, vitesse_max):
        # Attributs d'instance (propres à chaque objet)
        self.marque = marque
        self.vitesse_max = vitesse_max
        self._vitesse = 0      # attribut "protégé" (convention)

    def accelerer(self, v):
        if self._vitesse + v <= self.vitesse_max:
            self._vitesse += v

    def vitesse(self):
        return self._vitesse

    def __str__(self):
        return f"{self.marque} à {self._vitesse} km/h"

# Instanciation
ma_voiture = Voiture("Toyota", 180)
ma_voiture.accelerer(80)
print(ma_voiture)            # Toyota à 80 km/h
print(isinstance(ma_voiture, Voiture))  # True

👨‍👩‍👧 Héritage

L'héritage permet à une classe fille de réutiliser et spécialiser le comportement d'une classe parente.

class Animal:
    def __init__(self, nom):
        self.nom = nom
    def parler(self):
        return "..."
    def __str__(self):
        return f"{self.nom} dit : {self.parler()}"

class Chien(Animal):    # Chien hérite de Animal
    def parler(self):    # Redéfinition (override)
        return "Woof !"

class Chat(Animal):
    def __init__(self, nom, interieur):
        super().__init__(nom)  # appel du constructeur parent
        self.interieur = interieur
    def parler(self):
        return "Miaou !"

# Polymorphisme : même interface, comportement différent
animaux = [Chien("Rex"), Chat("Felix", True)]
for a in animaux:
    print(a)   # __str__ utilise parler() → polymorphisme

Getter / Setter

On contrôle l'accès aux attributs privés via des méthodes dédiées :

def get_solde(self): return self.__solde
def set_solde(self, v):
    if v >= 0: self.__solde = v

🚨

Pièges POO au bac

  • Oublier self comme premier paramètre de toute méthode → TypeError
  • Confondre attribut de classe et attribut d'instance
  • Oublier super().__init__() dans le constructeur de la classe fille
  • __str__ doit retourner une chaîne de caractères (pas afficher avec print)
  • Par défaut, == compare les identités (adresses mémoire), pas les valeurs — définir __eq__

Exo C1 — Classe Rectangle

Créer une classe Rectangle avec largeur et hauteur. Ajouter les méthodes aire(), perimetre() et est_carre().

class Rectangle:
    def __init__(self, largeur, hauteur):
        self.largeur = largeur
        self.hauteur = hauteur

    def aire(self):
        return self.largeur * self.hauteur

    def perimetre(self):
        return 2 * (self.largeur + self.hauteur)

    def est_carre(self):
        return self.largeur == self.hauteur

r = Rectangle(4, 6)
print(r.aire())       # → 24
print(r.perimetre()) # → 20
⭐⭐ Intermédiaire

Exo C2 — Classe Fraction

Créer une classe Fraction (numérateur, dénominateur) avec réduction automatique (pgcd) et méthodes __str__, __add__, __mul__.

from math import gcd

class Fraction:
    def __init__(self, num, den):
        if den == 0: raise ValueError("dénominateur nul")
        g = gcd(abs(num), abs(den))
        signe = 1 if den > 0 else -1
        self.num = signe * num // g
        self.den = signe * den // g

    def __str__(self):
        return f"{self.num}/{self.den}"

    def __add__(self, other):
        return Fraction(self.num*other.den + other.num*self.den,
                        self.den * other.den)

    def __mul__(self, other):
        return Fraction(self.num * other.num, self.den * other.den)

print(Fraction(1,3) + Fraction(1,6))  # → 1/2

Exo H1 — Formes géométriques

Créer une classe abstraite Forme avec aire() et perimetre(). Faire hériter Cercle, Rectangle et Triangle. Écrire une fonction plus_grande_aire(formes).

import math

class Forme:
    def aire(self): raise NotImplementedError
    def perimetre(self): raise NotImplementedError
    def __repr__(self):
        return f"{type(self).__name__}(aire={self.aire():.2f})"

class Cercle(Forme):
    def __init__(self, r): self.r = r
    def aire(self): return math.pi * self.r**2
    def perimetre(self): return 2 * math.pi * self.r

class Rectangle(Forme):
    def __init__(self, l, h): self.l = l; self.h = h
    def aire(self): return self.l * self.h
    def perimetre(self): return 2*(self.l+self.h)

def plus_grande_aire(formes):
    return max(formes, key=lambda f: f.aire())
⭐⭐⭐⭐ Niveau bac

Exo H2 — Compte bancaire avec héritage

Créer Compte (solde, depot, retrait) puis CompteEpargne(Compte) avec un taux d'intérêt et une méthode appliquer_interets(). Tester le polymorphisme.

class Compte:
    def __init__(self, solde=0):
        self._solde = solde

    def depot(self, montant):
        if montant > 0: self._solde += montant

    def retrait(self, montant):
        if 0 < montant <= self._solde: self._solde -= montant
        else: raise ValueError("solde insuffisant")

    def __str__(self):
        return f"Compte: {self._solde}€"

class CompteEpargne(Compte):
    def __init__(self, solde=0, taux=0.03):
        super().__init__(solde)
        self.taux = taux

    def appliquer_interets(self):
        self._solde *= (1 + self.taux)

    def __str__(self):
        return f"Épargne: {self._solde:.2f}€ (taux {self.taux*100}%)"

Exo H3 — Pile générique avec itérateur

Implémenter une classe Pile générique avec __len__, __iter__, __contains__ et __repr__.

class Pile:
    def __init__(self): self._data = []
    def empiler(self, x): self._data.append(x)
    def depiler(self): return self._data.pop()
    def est_vide(self): return not self._data

    def __len__(self): return len(self._data)
    def __iter__(self): return reversed(self._data)  # du sommet vers la base
    def __contains__(self, x): return x in self._data
    def __repr__(self):
        return f"Pile({list(reversed(self._data))})"

p = Pile()
p.empiler(1); p.empiler(2); p.empiler(3)
print(list(p))    # → [3, 2, 1]
print(2 in p)     # → True
print(len(p))     # → 3
