Graphes orientés BFS / DFS Dijkstra

🎯 Introduction

Un graphe est un ensemble de sommets (nœuds) reliés par des arêtes (arcs si orienté). Les graphes modélisent les réseaux routiers, sociaux, Internet, les dépendances entre tâches...

✅ Ce qu'il faut savoir

  • Deux représentations : matrice d'adjacence et liste d'adjacence
  • Différence graphe orienté / non orienté / pondéré
  • BFS (parcours en largeur) avec une file
  • DFS (parcours en profondeur) avec récursion ou pile
  • Algorithme de Dijkstra (plus courts chemins, poids positifs)
  • Notions : connexité, cycle, chemin

📐 Représentations

Chaque sommet est associé à la liste de ses voisins. Efficace pour les graphes creux.

# Graphe non orienté
graphe = {
    'A': ['B', 'C'],
    'B': ['A', 'D'],
    'C': ['A', 'D'],
    'D': ['B', 'C']
}

# Graphe pondéré orienté
graphe_p = {
    'A': [('B', 4), ('C', 2)],
    'B': [('D', 5)],
    'C': [('B', 1), ('D', 8)],
    'D': []
}

Matrice n×n : M[i][j] = 1 si arête de i vers j. Efficace pour les graphes denses.

# Graphe avec 4 sommets : 0,1,2,3
matrice = [
    [0, 1, 1, 0],   # 0 → 1, 0 → 2
    [1, 0, 0, 1],   # 1 → 0, 1 → 3
    [1, 0, 0, 1],   # 2 → 0, 2 → 3
    [0, 1, 1, 0]    # 3 → 1, 3 → 2
]
# Tester l'arête (0, 1) :
matrice[0][1] == 1   # True — O(1)

🔄 Parcours : BFS et DFS

from collections import deque

# BFS — Parcours en Largeur (Breadth-First Search)
# Utilise une FILE. Trouve le plus court chemin (non pondéré).
def bfs(graphe, depart):
    visites = {depart}
    file = deque([depart])
    while file:
        s = file.popleft()
        print(s)
        for voisin in graphe[s]:
            if voisin not in visites:
                visites.add(voisin)
                file.append(voisin)

# DFS — Parcours en Profondeur (Depth-First Search)
# Utilise la RÉCURSION (ou une pile). Explore le plus loin possible.
def dfs(graphe, s, visites=None):
    if visites is None: visites = set()
    visites.add(s)
    print(s)
    for voisin in graphe[s]:
        if voisin not in visites:
            dfs(graphe, voisin, visites)
    return visites

🗺️ Algorithme de Dijkstra

⚠️

Condition d'utilisation

Dijkstra ne fonctionne qu'avec des poids positifs ou nuls. Pour les poids négatifs, utiliser Bellman-Ford.

import heapq

def dijkstra(graphe, depart):
    # graphe[s] = [(voisin, poids), ...]
    dist = {s: float('inf') for s in graphe}
    dist[depart] = 0
    tas = [(0, depart)]
    while tas:
        d, u = heapq.heappop(tas)
        if d > dist[u]: continue  # entrée obsolète
        for v, poids in graphe[u]:
            nd = dist[u] + poids
            if nd < dist[v]:
                dist[v] = nd
                heapq.heappush(tas, (nd, v))
    return dist

🚨

Pièges du bac

  • Oublier de marquer les sommets comme visités → boucle infinie
  • Confondre BFS (file) et DFS (pile/récursion)
  • Dijkstra avec poids négatifs → résultats faux
  • Pour un graphe non orienté, chaque arête apparaît deux fois dans la liste d'adjacence

⭐ Facile

Exo R1 — Convertir liste → matrice

Écrire liste_vers_matrice(g, n) qui convertit un graphe non orienté représenté par liste d'adjacence en matrice d'adjacence (n sommets numérotés 0 à n-1).

def liste_vers_matrice(g, n):
    mat = [[0] * n for _ in range(n)]
    for u in g:
        for v in g[u]:
            mat[u][v] = 1
            mat[v][u] = 1  # non orienté
    return mat

g = {0:[1,2], 1:[0,3], 2:[0], 3:[1]}
# mat[0][1]=1, mat[0][2]=1, mat[1][3]=1 ...
⭐⭐ Intermédiaire

Exo R2 — Degré d'un sommet

Écrire degre(g, s) qui retourne le degré du sommet s (nombre de voisins) dans un graphe en liste d'adjacence. Trouver aussi le sommet de degré maximal.

def degre(g, s):
    return len(g.get(s, []))

def sommet_max_degre(g):
    return max(g, key=lambda s: len(g[s]))

g = {'A':['B','C','D'], 'B':['A'], 'C':['A'], 'D':['A']}
print(sommet_max_degre(g))  # → 'A' (degré 3)

🔍 BFS — Parcours en largeur

⭐ Facile

Exo B1 — BFS basique

Écrire bfs(g, depart) qui retourne la liste des sommets visités en ordre BFS depuis depart.

from collections import deque

def bfs(g, depart):
    vus = {depart}
    file = deque([depart])
    ordre = []
    while file:
        s = file.popleft()
        ordre.append(s)
        for v in g.get(s, []):
            if v not in vus:
                vus.add(v)
                file.append(v)
    return ordre
⭐⭐ Intermédiaire

Exo B2 — Plus court chemin (BFS)

Écrire plus_court_chemin(g, src, dst) qui retourne la liste des sommets du plus court chemin entre src et dst dans un graphe non pondéré. Retourner None si inexistant.

from collections import deque

def plus_court_chemin(g, src, dst):
    predecesseur = {src: None}
    file = deque([src])
    while file:
        s = file.popleft()
        if s == dst:
            # reconstruire le chemin
            chemin = []
            while s is not None:
                chemin.append(s); s = predecesseur[s]
            return chemin[::-1]
        for v in g.get(s, []):
            if v not in predecesseur:
                predecesseur[v] = s
                file.append(v)
    return None
⭐⭐⭐ Avancé

Exo B3 — Composantes connexes

Écrire composantes_connexes(g) qui retourne la liste des composantes connexes (chaque composante = liste de sommets).

from collections import deque

def composantes_connexes(g):
    vus = set()
    composantes = []
    for depart in g:
        if depart in vus: continue
        # BFS depuis ce sommet non visité
        comp = []
        file = deque([depart])
        vus.add(depart)
        while file:
            s = file.popleft()
            comp.append(s)
            for v in g.get(s, []):
                if v not in vus:
                    vus.add(v); file.append(v)
        composantes.append(comp)
    return composantes

🌀 DFS — Parcours en profondeur

⭐ Facile

Exo D1 — DFS récursif

Écrire dfs(g, depart) qui retourne la liste des sommets visités en DFS (version récursive).

def dfs(g, depart, vus=None):
    if vus is None: vus = set()
    vus.add(depart)
    ordre = [depart]
    for v in g.get(depart, []):
        if v not in vus:
            ordre += dfs(g, v, vus)
    return ordre
⭐⭐⭐⭐ Niveau bac

Exo D2 — Détection de cycle

Écrire a_un_cycle(g) qui détecte si un graphe non orienté contient un cycle, en utilisant DFS.

def a_un_cycle(g):
    vus = set()
    def dfs(s, parent):
        vus.add(s)
        for v in g.get(s, []):
            if v not in vus:
                if dfs(v, s): return True
            elif v != parent:  # voisin déjà vu ≠ parent → cycle
                return True
        return False
    for s in g:
        if s not in vus:
            if dfs(s, None): return True
    return False
⭐⭐⭐⭐⭐ Expert

Exo D3 — Tri topologique

Dans un DAG (graphe orienté acyclique), le tri topologique ordonne les sommets de sorte que chaque arc va de gauche à droite. Implémenter avec DFS.

def tri_topologique(g):
    """g : dict {sommet: [voisins orientés]}"""
    vus = set()
    resultat = []
    def dfs(s):
        vus.add(s)
        for v in g.get(s, []):
            if v not in vus: dfs(v)
        resultat.append(s)  # ajouter APRÈS les successeurs
    for s in g:
        if s not in vus: dfs(s)
    return resultat[::-1]  # inverser

g = {'A':['C'], 'B':['C','D'], 'C':['E'], 'D':['F'], 'E':['F'], 'F':[]}
print(tri_topologique(g))  # ['B', 'A', 'D', 'C', 'E', 'F'] (ou similaire)

🗺️ Dijkstra

⭐⭐⭐ Avancé

Exo Dij1 — Dijkstra complet

Implémenter l'algorithme de Dijkstra qui retourne les distances minimales depuis une source dans un graphe pondéré à poids positifs.

import heapq

def dijkstra(g, src):
    """g : {sommet: [(voisin, poids),...]}"""
    dist = {s: float('inf') for s in g}
    dist[src] = 0
    tas = [(0, src)]  # (distance, sommet)
    while tas:
        d, s = heapq.heappop(tas)
        if d > dist[s]: continue  # déjà mieux
        for v, poids in g.get(s, []):
            nd = dist[s] + poids
            if nd < dist[v]:
                dist[v] = nd
                heapq.heappush(tas, (nd, v))
    return dist

g = {'A':[('B',4),('C',2)], 'B':[('D',3)],
     'C':[('B',1),('D',5)], 'D':[]}
print(dijkstra(g, 'A'))
# → {'A':0, 'B':3, 'C':2, 'D':6}
⭐⭐⭐⭐⭐ Expert

Exo Dij2 — Dijkstra avec chemin

Modifier Dijkstra pour qu'il retourne aussi le chemin optimal (liste de sommets) depuis la source vers chaque destination.

import heapq

def dijkstra_chemin(g, src):
    dist = {s: float('inf') for s in g}
    pred = {s: None for s in g}
    dist[src] = 0
    tas = [(0, src)]
    while tas:
        d, s = heapq.heappop(tas)
        if d > dist[s]: continue
        for v, poids in g.get(s, []):
            nd = dist[s] + poids
            if nd < dist[v]:
                dist[v] = nd; pred[v] = s
                heapq.heappush(tas, (nd, v))

    def chemin(dst):
        c = []
        while dst is not None:
            c.append(dst); dst = pred[dst]
        return c[::-1]

    return dist, chemin

📌 Fiche synthèse

REPRÉSENTATIONS

  • Liste adj : O(V+E) mémoire
  • Matrice : O(V²) mémoire
  • Matrice → test arête O(1)
  • Liste → voisins O(degré)

PARCOURS

  • BFS : file, plus court chemin
  • DFS : récursion / pile
  • Complexité : O(V + E)
  • Toujours marquer les visités !

DIJKSTRA

  • Poids positifs uniquement
  • Utilise un tas-min (heapq)
  • O((V+E) log V)
  • Plus court depuis source
