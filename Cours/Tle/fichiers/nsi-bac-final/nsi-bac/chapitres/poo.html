Classes Héritage Encapsulation

🎯 Introduction

La POO est un paradigme qui organise le code autour d'objets combinant des données (attributs) et des comportements (méthodes). Python est un langage multi-paradigme orienté objet.

✅ Ce qu'il faut savoir

  • Créer une classe avec __init__, self, attributs et méthodes
  • Héritage : class Enfant(Parent), super()
  • Polymorphisme : même méthode, comportement différent selon le type
  • Encapsulation : convention _attribut (protégé), __attribut (privé)
  • Méthodes spéciales : __str__, __repr__, __len__, __eq__

🏗️ Classes et objets

class Voiture:
    # Attribut de classe (partagé par toutes les instances)
    nb_roues = 4

    def __init__(self, marque, vitesse_max):
        # Attributs d'instance (propres à chaque objet)
        self.marque = marque
        self.vitesse_max = vitesse_max
        self._vitesse = 0      # attribut "protégé" (convention)

    def accelerer(self, v):
        if self._vitesse + v <= self.vitesse_max:
            self._vitesse += v

    def vitesse(self):
        return self._vitesse

    def __str__(self):
        return f"{self.marque} à {self._vitesse} km/h"

# Instanciation
ma_voiture = Voiture("Toyota", 180)
ma_voiture.accelerer(80)
print(ma_voiture)            # Toyota à 80 km/h
print(isinstance(ma_voiture, Voiture))  # True

👨‍👩‍👧 Héritage

L'héritage permet à une classe fille de réutiliser et spécialiser le comportement d'une classe parente.

class Animal:
    def __init__(self, nom):
        self.nom = nom
    def parler(self):
        return "..."
    def __str__(self):
        return f"{self.nom} dit : {self.parler()}"

class Chien(Animal):    # Chien hérite de Animal
    def parler(self):    # Redéfinition (override)
        return "Woof !"

class Chat(Animal):
    def __init__(self, nom, interieur):
        super().__init__(nom)  # appel du constructeur parent
        self.interieur = interieur
    def parler(self):
        return "Miaou !"

# Polymorphisme : même interface, comportement différent
animaux = [Chien("Rex"), Chat("Felix", True)]
for a in animaux:
    print(a)   # __str__ utilise parler() → polymorphisme

Getter / Setter

On contrôle l'accès aux attributs privés via des méthodes dédiées :

def get_solde(self): return self.__solde
def set_solde(self, v):
    if v >= 0: self.__solde = v

🚨

Pièges POO au bac

  • Oublier self comme premier paramètre de toute méthode → TypeError
  • Confondre attribut de classe et attribut d'instance
  • Oublier super().__init__() dans le constructeur de la classe fille
  • __str__ doit retourner une chaîne de caractères (pas afficher avec print)
  • Par défaut, == compare les identités (adresses mémoire), pas les valeurs — définir __eq__

🏋️ Exercices

FacileClasse Fraction

Créer une classe Fraction avec numérateur et dénominateur. Implémenter __str__ et __add__.

✅ Correction
from math import gcd
class Fraction:
    def __init__(self, num, den):
        if den == 0: raise ValueError("Dénominateur nul")
        g = gcd(abs(num), abs(den))
        self.num = num // g
        self.den = den // g
    def __str__(self):
        return f"{self.num}/{self.den}"
    def __add__(self, other):
        return Fraction(self.num*other.den + other.num*self.den,
                       self.den * other.den)
Niveau bacHéritage : Compte bancaire

Créer une classe Compte avec solde et méthodes depot/retrait. Puis une classe CompteEpargne qui hérite de Compte et ajoute un taux d'intérêt et une méthode capitaliser.

✅ Correction
class Compte:
    def __init__(self, titulaire, solde=0):
        self.titulaire = titulaire
        self.__solde = solde
    def solde(self): return self.__solde
    def depot(self, m): self.__solde += m
    def retrait(self, m):
        if m > self.__solde: raise ValueError("Solde insuffisant")
        self.__solde -= m
    def __str__(self): return f"{self.titulaire}: {self.__solde}€"

class CompteEpargne(Compte):
    def __init__(self, titulaire, solde, taux):
        super().__init__(titulaire, solde)
        self.taux = taux
    def capitaliser(self):
        self.depot(self.solde() * self.taux)
